### OpenClaw-RL: Train Any Agent Simply by Talking

- **Paper ID**: 2603.10165v1
- **URL**: https://arxiv.org/abs/2603.10165v1

OpenClaw-RL论文提出了一种新颖的强化学习框架，旨在解决现有强化学习系统无法充分利用交互过程中生成的下一状态信号的问题。研究的核心动机在于，用户回复、工具输出、终端或GUI状态变化等下一状态信号是普遍存在的，但现有系统未能将其作为实时在线学习的资源加以利用。论文的目标是通过统一的框架，将这些不同形式的交互信号整合用于策略训练。

该方法的核心创新在于提出了两种信息提取机制：评估信号和指引信号。评估信号通过PRM（评估模型）提取为标量奖励，用于衡量动作表现；指引信号通过“基于回溯的策略蒸馏”（Hindsight-Guided On-Policy Distillation, OPD）机制，从下一状态中提取文本提示，构建增强的教师上下文，并提供比标量奖励更丰富的方向性监督。此外，框架采用异步设计，使模型能够在服务实时请求的同时，PRM进行交互评估，训练器更新策略，且无需额外协调开销。

实验结果表明，OpenClaw-RL在多个场景中表现出色，包括个性化代理的对话信号学习（如用户纠正和反馈）以及通用代理在终端、GUI、软件工程任务和工具调用中的强化学习。此外，研究还展示了过程奖励在这些场景中的实用性。

与现有方法相比，OpenClaw-RL的最大区别在于其统一性和实时学习能力。它打破了不同交互类型之间的隔阂，将所有交互视为统一的训练资源，并通过异步架构实现高效的在线更新。

该研究的潜在影响深远，不仅为个性化代理提供了自我改进的能力，还为通用人工智能系统的扩展性和适应性奠定了基础。未来方向包括进一步优化评估模型和蒸馏机制，以及扩展至更复杂的多模态交互场景。

### PRECEPT: Planning Resilience via Experience, Context Engineering & Probing Trajectories A Unified Framework for Test-Time Adaptation with Compositional Rule Learning and Pareto-Guided Prompt Evolution

- **Paper ID**: 2603.09641v1
- **URL**: https://arxiv.org/abs/2603.09641v1

PRECEPT是一种针对测试时适应性的统一框架，旨在解决大语言模型（LLM）在知识存储、规则组合以及对过时或敌对知识检测方面的局限性。研究的动机在于提升模型在复杂条件下的检索能力、规则组合的可靠性以及动态环境中的适应性。

该框架由三个核心组件构成：(1)基于结构化条件键的确定性精确匹配规则检索，避免部分匹配导致的解释错误；(2)基于贝叶斯源可靠性和阈值规则失效的冲突感知记忆模块，用于处理静态与动态知识的冲突并适应漂移；(3)COMPASS模块，通过帕累托引导的提示进化外循环优化提示质量。这些组件协同工作，支持语义层级规则堆叠、知识冲突解决及动态环境中的持续学习。

实验结果表明，PRECEPT在多个关键指标上显著优于现有方法，包括首次尝试成功率提升41.1个百分点、规则组合泛化能力提升33.3个百分点、漂移恢复能力提升55个百分点等。同时，该框架在敌对静态知识环境下表现出强鲁棒性，并显著减少了决策步骤。

与现有方法相比，PRECEPT通过精确检索和冲突感知记忆显著减少了错误率，并通过帕累托优化实现提示进化，提升了模型的动态适应性和规则组合能力。其创新点在于将规则检索、冲突处理和提示优化有机结合，形成一个统一的适应性框架。

该研究为提升LLM在动态复杂任务中的适应性提供了新的思路，未来可进一步探索框架在更多任务场景中的扩展性及与其他AI系统的集成潜力。

### AttriGuard: Defeating Indirect Prompt Injection in LLM Agents via Causal Attribution of Tool Invocations

- **Paper ID**: 2603.10749v1
- **URL**: https://arxiv.org/abs/2603.10749v1

AttriGuard: Defeating Indirect Prompt Injection in LLM Agents via Causal Attribution of Tool Invocations是一篇针对大型语言模型（LLM）代理中间接提示注入攻击（IPI）问题的研究论文。研究的动机源于IPI攻击的严重性，该攻击通过在工具输出中嵌入恶意指令，劫持代理的执行过程，现有防御方法对未见过的攻击负载往往缺乏泛化能力。论文提出了一种全新的防御范式——基于工具调用的因果归因，通过分析工具调用的因果关系来区分用户意图驱动的调用与受不可信观察驱动的调用。

核心方法是AttriGuard系统，该系统利用并行反事实测试来验证每次工具调用的必要性。技术创新包括教师强制的影子重放防止归因混淆、分层控制衰减以压制多样化控制通道同时保留任务相关信息，以及对LLM随机性具有鲁棒性的模糊生存标准。在实验中，AttriGuard在四种LLM和两个代理基准测试中表现出色，在静态攻击下实现了0%的攻击成功率，同时保持了任务效用的微小损失和适度的计算开销。此外，AttriGuard在自适应优化攻击下依然具有显著的防御能力，而现有主流防御方法在此场景中性能显著下降。

与现有方法相比，AttriGuard从因果关系出发，突破了传统输入级语义判别的局限性，提供了更强的泛化能力和鲁棒性。该研究为LLM代理的安全性提供了新的方向，未来可以进一步探索因果归因方法在更复杂任务场景中的应用，以及优化系统效率以适应实时部署需求。

### Can Large Language Models Keep Up? Benchmarking Online Adaptation to Continual Knowledge Streams

- **Paper ID**: 2603.07392
- **URL**: https://arxiv.org/abs/2603.07392

该论文研究了大型语言模型（LLMs）在动态、持续变化的知识流中进行在线适应的能力。随着现实世界知识的不断演化和增量更新，模型需要能够实时适应新信息，以保持准确性和有效性。为此，作者提出了一个名为OAKS（Online Adaptation to Continual Knowledge Streams）的基准，用于评估模型在流式、持续更新的知识环境中的在线适应能力。

OAKS基准由一系列细粒度的上下文块组成，其中事实会随着时间动态变化。基准包含两个数据集：OAKS-BABI和OAKS-Novel，分别模拟了事实在不同上下文块中的多次演变。这些数据集提供了密集的标注，用于精确评估模型是否能够正确追踪变化。作者对14种不同推理方法的模型进行了评估，发现现有方法在OAKS基准上的表现存在显著局限性。即使是最先进的模型和具备记忆功能的代理系统，也难以在流式环境中实现稳健的适应，表现出状态跟踪延迟和对干扰的敏感性。

该研究的核心创新在于提出了一个系统化的评估框架，专注于模型在动态知识流中的适应能力，而非传统静态任务。此外，OAKS基准通过动态变化的上下文设计，模拟了真实世界中知识演化的复杂性，这在现有基准中尚属首次。

实验结果表明，当前模型在动态知识环境中的适应能力远未达到理想水平，揭示了在线学习和记忆机制的关键挑战。该研究为未来改进语言模型的动态适应能力提供了重要方向，例如开发更高效的记忆管理机制和实时状态更新方法。

这项工作区别于以往静态知识评估方法，专注于动态环境下的实时适应能力，填补了这一领域的研究空白。其潜在影响包括推动LLMs在新闻更新、实时对话和动态领域知识处理等实际应用中的表现改进，同时为持续学习和在线适应研究奠定了新的基准和方向。

### Test-Driven AI Agent Definition (TDAD): Compiling Tool-Using Agents from Behavioral Specifications

- **Paper ID**: 2603.08806
- **URL**: https://arxiv.org/abs/2603.08806

Test-Driven AI Agent Definition (TDAD) 提出了一种新颖的方法，用以解决工具型大语言模型（LLM）代理在生产环境中行为一致性难以量化的问题。当前开发实践中，提示词的微小改动可能导致隐性回归，工具误用和策略违规常在部署后才被发现。为此，本文提出了一种基于行为规范的代理开发方法，将代理提示词视为可编译工件，通过行为规范生成可执行测试，并通过迭代优化提示词以通过测试。

TDAD 的核心技术创新包括三项机制：(1) 可见/隐藏测试拆分，在编译过程中隐藏部分评估测试以防止规范投机；(2) 语义变异测试，通过后续代理生成潜在有缺陷的提示词变体，并利用测试套件评估其检测能力；(3) 规范演化场景，用于量化需求变更时的回归安全性。这些机制有效地提升了代理开发的可靠性和行为一致性。

实验基于 SpecSuite-Core 基准测试，涵盖四类深度规范化代理（政策合规、数据分析、操作手册执行和确定性行为）。在 24 次独立试验中，TDAD 在初始规范编译中达成 92% 的成功率，隐藏测试通过率平均为 97%；在规范演化后的编译中，成功率为 58%，大部分失败案例仅在 1-2 个可见测试上失效，同时变异分数达 86-100%，隐藏测试通过率为 78%，回归安全性评分为 97%。

与现有方法相比，TDAD 不仅在行为一致性和规范化测试上更具系统性，还通过创新的测试机制有效减少了规范投机和回归风险。这项工作为工具型 LLM 代理的开发提供了更高的可靠性和可控性，未来可进一步扩展到更多复杂场景和多模态代理开发中。
