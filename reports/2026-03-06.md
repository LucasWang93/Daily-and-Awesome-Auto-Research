## Archived Papers - Deep Dive

### Mozi: Governed Autonomy for Drug Discovery LLM Agents

- **Paper ID**: 2603.03655
- **URL**: https://arxiv.org/abs/2603.03655

Mozi是一种针对药物发现的大型语言模型（LLM）代理架构，旨在解决当前工具增强型LLM在高风险领域（如药物研发）中面临的两大核心问题：工具使用缺乏治理以及长周期任务的可靠性不足。现有方法在药物研发的复杂流程中易因早期错误（如幻觉生成）导致后续不可控的失败。为此，Mozi提出了一种双层架构：控制层（Control Plane）通过监督-工作者的角色分离，限制工具使用范围，并引入基于反思的重新规划机制；工作流层（Workflow Plane）将药物研发的关键阶段（如靶点识别、先导化合物优化）模块化为状态化、可组合的技能图，同时引入严格的数据契约和关键决策点的人机协作机制，确保科学严谨性。

Mozi基于“自由推理用于低风险任务，结构化执行用于长周期流程”的设计原则，内置鲁棒性机制和可追溯性，显著降低错误累积风险。在PharmaBench基准测试中，Mozi展示了优于现有基线的任务协调精度；通过端到端的药物开发案例研究，Mozi成功探索了大规模化学空间，严格执行毒性筛选，并生成了具有竞争力的候选化合物，展现了从“脆弱对话者”到“可靠共科学家”的转变。

与现有方法相比，Mozi在工具治理和长周期可靠性方面有显著创新，特别是在高不确定性决策中引入了人机协作和数据约束。未来，该方法可进一步拓展至其他科学领域，推动AI在复杂科学问题中的应用与可信性提升。

### Memex(RL): Scaling Long-Horizon LLM Agents via Indexed Experience Memory

- **Paper ID**: 2603.04257
- **URL**: https://arxiv.org/abs/2603.04257

本文提出了一种新的机制Memex及其强化学习框架MemexRL，旨在解决大语言模型（LLM）代理在处理长时间跨度任务时因有限上下文窗口而面临的瓶颈问题。现有方法通过截断或摘要压缩上下文，但这些方法不可避免地丢失部分信息，影响模型决策质量。为此，作者设计了Memex机制，通过索引化的经验记忆库，将完整的交互信息存储在外部数据库，同时在工作上下文中保留简洁的结构化摘要和稳定索引。模型可以根据任务需求动态检索索引，恢复所需的原始证据。

MemexRL进一步优化了写入和读取行为，通过奖励函数设计引导代理学习如何摘要、归档、索引以及检索，从而在有限上下文预算下实现高效记忆管理。理论分析表明，Memex机制能够在保持计算效率的同时，通过有限的索引检索保证决策质量。实验结果显示，基于MemexRL训练的代理在长时间跨度任务中显著提高了任务成功率，同时显著减少了工作上下文的长度。

与现有方法相比，Memex通过索引化存储和动态检索避免了信息丢失，提供了一种更少损耗的长时记忆解决方案。这项研究为长时间跨度任务中的LLM代理提供了新的技术路径，未来可进一步探索其在复杂任务中的应用潜力以及与其他记忆机制的结合。
