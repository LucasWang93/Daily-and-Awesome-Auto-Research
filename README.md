# Awesome Auto-Research

An English-first curated knowledge base for agentic research, AI scientist systems, literature-review agents, and related automation for scientific work.

## What Is Auto-Research?

Auto-research refers to systems that help automate parts of the research loop: literature discovery, synthesis, hypothesis generation, experiment planning, implementation, evaluation, and reporting. This repository is intentionally selective. It tracks the most important repositories, landmark papers, and recent additions that matter for building or understanding research agents.

## Workflow

```bash
python run.py ingest
python run.py build-readme
python run.py sync-index
python run.py curate-report
```

The default daily job is `python run.py ingest`, which collects new papers from ArXiv and HuggingFace, archives high-signal paper cards, refreshes structured indexes in `data/`, updates curated README sections, and writes a digest in `reports/`.

## Most Important GitHub Repos

<!-- BEGIN: curated-repos -->
- [The AI Scientist](https://github.com/SakanaAI/AI-Scientist) [landmark]: An end-to-end prototype for automated research idea generation, experimentation, and paper writing. Why it matters: It made the concept of autonomous research workflows concrete for a broad technical audience. Relation to auto-research: A reference point for agentic systems that attempt to complete more than one stage of the research loop. Representative reference: The AI Scientist paper and project release.
- [OpenHands](https://github.com/All-Hands-AI/OpenHands) [active]: A general software agent platform with strong implications for experiment and research automation. Why it matters: It contributes reusable infrastructure for tool-using agents that can support research tasks. Relation to auto-research: Useful for building paper-to-code and experiment-execution workflows. Representative reference: OpenHands benchmark demos and agent framework.
- [gpt-researcher](https://github.com/assafelovic/gpt-researcher) [active]: A focused research assistant for multi-source web research and report generation. Why it matters: It is one of the clearest open implementations of literature-style autonomous synthesis. Relation to auto-research: Relevant to literature-review agents and retrieval-driven research workflows. Representative reference: GPT Researcher open-source project and docs.
